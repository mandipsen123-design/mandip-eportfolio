Module 05 — The Language

JavaScript

The only programming language native to the browser. JavaScript adds interactivity, logic, and dynamic behaviour to your HTML and CSS — making pages come alive.

Fundamentals

Core Concepts

JavaScript is a high-level, interpreted language with dynamic typing. These are the building blocks every JS developer uses daily.

📦

Variables

let, const, var — store values that can be referenced and manipulated.

🔢

Data Types

String, Number, Boolean, Array, Object, null, undefined — the kinds of values JS can work with.

⚡

Functions

Reusable blocks of code. Arrow functions (=>) are the modern syntax for concise definitions.

🔀

Control Flow

if/else, for, while — logic that determines what code runs under what conditions.

🌲

DOM

The Document Object Model — JS's interface to read and modify HTML elements in real time.

👂

Events

React to user actions — clicks, keypresses, scrolls — by attaching event listeners.

⏳

Async / Await

Handle asynchronous operations (API calls, file reads) without blocking the main thread.

📡

Fetch API

Request data from servers and update the page without a full reload — the foundation of modern SPAs.

variables & types const name = "Priya"; // String let score = 98; // Number const isAdmin = true; // Boolean const skills = ["HTML", "CSS", "JS"]; // Array const user = { name, score, isAdmin }; // Object

Event Reference

Common Event Types

JavaScript listens to user interactions using addEventListener(). Every interaction type has a named event.

🖱️ click

Mouse button pressed and released on an element.

⌨️ keydown

Any keyboard key is pressed — check event.key.

📝 input

Value of an input field changes (every keystroke).

📤 submit

A form is submitted — use preventDefault().

🖱️ mouseover

Mouse cursor enters an element's bounding box.

📜 scroll

User scrolls the page — useful for scroll animations.

📦 load

Page (or image/script) has fully loaded.

🔁 change

Value of a select/checkbox/radio changes.
