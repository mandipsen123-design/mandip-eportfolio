Module 03 — Presentation

CSS

Cascading Style Sheets transform plain HTML structure into beautiful, responsive, animated interfaces. CSS controls every visual property — colour, spacing, layout, and motion.

CSS Syntax

Anatomy of a CSS Rule

A CSS rule targets an element with a selector, then applies declarations — each a property–value pair — inside curly braces.

h1 { /* selector targets all <h1> elements */ color: #21d4ff; /* property: value */ font-size: 2.5rem; font-weight: 700; margin-bottom: 1rem; text-align: center; }
Selector — targets HTML elements
Property — what to style
Value — how to style it

Layout Fundamentals

The Box Model

Every HTML element is a rectangular box. CSS controls four layers around the content: padding, border, and margin. Understanding this is essential for precise layout control.

MARGIN
BORDER
PADDING
CONTENT
Margin
Space outside the element — separates it from neighbours
Border
Optional line around the element — can be coloured, styled
Padding
Space inside the element, between content and border
Content
The actual text, image, or child elements

CSS Visual Techniques

The same HTML structure, four completely different CSS treatments. This is the power of separating content from presentation.

box-shadow

Layered shadows create depth and elevation — the material design technique.

linear-gradient

CSS gradients blend multiple colours — no images needed, fully scalable.

backdrop-filter

Glassmorphism uses blur and transparency to create frosted-glass panels.

border-only

A minimal outline style — clean, high-contrast, no background fill.
