Module 04 — Framework

Bootstrap 5

The world's most-used CSS framework. Pre-built components, a responsive 12-column grid, and hundreds of utility classes — ship UIs in a fraction of the time.

What it is

CSS you don't have to write

Bootstrap provides a standardised set of CSS classes you apply directly to HTML. Instead of writing display:flex; align-items:center; gap:1rem; from scratch, you write d-flex align-items-center gap-3.

It enforces consistency, speeds up development, and is mobile-first by default — meaning styles apply to small screens first, then expand up.


<style>
  .card { border-radius: 8px; padding: 1.5rem;
          box-shadow: 0 2px 8px rgba(0,0,0,.1); }
  .btn  { padding: .5rem 1rem; border-radius: 6px;
          background: #0d6efd; color: #fff; }
</style>


<div class="card p-4 shadow-sm rounded-3">
  <button class="btn btn-primary">Click me</button>
</div>
The Grid System

12 Columns, Infinite Layouts
